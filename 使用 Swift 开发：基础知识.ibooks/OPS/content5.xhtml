运算符是使代码运行的符号。使用运算符可以执行检查值、更改值或合并值等操作。运算符的用途包括处理和比较数字，检查 true 和 false 值，以及简化条件赋值等等。

在本节课中，你将学习 Swift 语言中的一些运算符，包括在加法 (+)、减法 (–)、乘法 (*) 和除法 (/) 中使用的基本数学运算符。

  • 如何将两种不同类型的数字相加
  • 如何找到除法运算的余数

  • Compound assignment (复合赋值)
  • Operator (运算符)

在编程中，运算符通过对其左右两侧的值执行操作来让代码工作。在阅读代码时，如果看到了不熟悉的符号，这个符号很可能就是运算符。

这一节课涵盖了最常见的运算符。

赋值

使用 = 运算符来赋值。将右侧的值赋给左侧的名称。

下面的代码将“Luke”声明为你最喜欢的人：

 

let favoritePerson = “Luke”

 

= 运算符也可用于修改或重新赋值。

以下代码声明了一个 shoeSize 变量，并将 8 赋值给这个变量。然后又将该值修改为 9：

 

var shoeSize = 8

shoeSize = 9 // 将 shoeSize 重新设为 9

 

基本算术

你可以使用 +、-、* 和 / 运算符来执行基本的数学运算。

 

var opponentScore = 3 * 8

// opponentScore 的值为 24

var myScore = 100 / 4

// myScore 的值为 25

 

你可以使用运算符来执行使用其他变量值的算术：

var totalScore = opponentScore + myScore// totalScore 的值为 49

 

运算符甚至可以引用当前变量，将其更新为新值：

 

myScore = myScore + 3 // myScore 的值更新为当前值加 3

 

针对带小数点的精度，可以对 Double 值执行相同的运算：

 

let totalDistance = 3.9

var distanceTraveled = 1.2

var remainingDistance = totalDistance - distanceTraveled

// remainingDistance 的值为 2.7

 

如果对 Int 值使用除法运算符 (/)，则结果将是向下取整到最接近的整数 Int 值，因为 Int 类型支持整数：

 

let x = 51

let y = 4

let z = x / y // z 的值为 12

 

如果你明确地将常量或变量声明为 Double 值，那么结果将是带小数点的值。

 

let x: Double = 51

let y: Double = 4

let z = x / y // z 的值为 12.75

 

如果代码需要带小数点的精度，则请务必使用 Double 值。

复合赋值

前面的代码片段通过将自身与数字相加来更新变量：

 

var myScore = 10

myScore = myScore + 3

// myScore 的值更新为当前值加 3

 

但还可以使用更好的方法来修改已赋值。你可以使用复合赋值运算符，即在算术运算符之后添加 = 运算符。

 

myScore += 3 // myScore 的值加 3

myScore -= 5 // myScore 的值减 5

myScore *= 2 // myScore 的值乘以 2

myScore /= 2 // myScore 的值除以 2

 

复合赋值运算符可以帮助你编写更干净、更简洁的代码。

余数运算符

使用余数运算符 (%) 可快速计算两个 Int 值相除的余数。

 

let dividend = 10

let divisor = 3

let quotient = dividend / divisor // quotient 的值为 3

let remainder = dividend % divisor // remainder 的值为 1

 

可以使用 dividend - (quotient * divisor) 来计算余数，但使用余数运算符更容易、更快速、更整洁、更简练。

运算顺序

数学运算始终遵循特定顺序。乘法和除法的优先级高于加法和减法，而括号的优先级则高于以上四种算法。

例如以下变量。

 

var x = 2

var y = 3

var z = 5

 

还有以下计算：

 

x + y * z // 值为 17

(x + y) * z // 值为 25

 

在上面的第一行中，乘法的优先级高于加法。在第二行中，系统会首先执行括号中的运算。

余数运算符的优先级与乘法和除法相同：

y + z % x // 值为 4

(y + z) % x // 值为 0

 

数值型类型转换

正如你所学到的那样，在执行数学运算时不能混合搭配数字类型。

例如，以下代码将产生编译错误，因为 x 是 Int 值，而 y 是 Double 值：

 

let x = 3

let y = 0.1415927

let pi = x + y

 

若要完成代码，可以将 x 创建为新的 Double 值，方法是为其加上你希望转换到的目标类型的前缀：

 

let x = 3

let y = 0.1415927

let pi = Double(x) + y

 

在修改后的代码中，Double(x) 利用 Int 值 x 创建了新的 Double 值，让编译器可以将这个新值与 y 相加，并将结果赋给 pi。
