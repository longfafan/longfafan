你可能会想到很多在达到某个条件之前需要持续执行的日常任务。例如，如果水瓶没有装满，你会继续向瓶中添水；或者如果作业没有完成，你会继续写作业。

这种需要完成并且重复执行任务的场景，编写代码时可以通过使用循环来实现。在本节课中，你将学习如何在 Swift 中创建循环、控制循环的条件，以及指定何时停止循环。

内容提要

  • 如何使用 for 循环逐个执行集合中的每个值
  • 如何迭代一个范围内的值
  • 如何编写某个条件不再为 true 时即停止继续执行的循环

词汇

  • Closed range operator (闭区间运算符)
  • for-in loop (for-in 循环)
  • Half-open range operator (半开区间运算符)
  • Iteration (迭代)
  • repeat-while loop (repeat-while 循环)
  • while loop (while 循环)

计算机程序擅长重复执行操作。开发者经常编写代码来跨多个对象执行同一操作、多次执行一项任务，或者持续执行工作直到满足特定条件才停止。Swift 提供了三种不同的方法来循环遍历或重复执行代码块。

 

For 循环

你将学习的第一个循环为 for 循环，更具体地说也称为 for-in 循环。在需要将某些操作重复执行固定的次数或遍历集合中的值时，使用 for 循环是非常适合的。

for-in 循环会针对某个范围、序列或集合中的每个项目执行一组语句。假设你希望针对 1 到 5 这个范围内的数字打印所有值。此时无需编写五条 print 语句，只需针对该范围使用 for-in 循环并编写一条 print 语句即可。具体语法如下所示：

 

for index in 1...5 {

print(”This is number \(index)”)

}

控制台输出：

This is number 1

This is number 2

This is number 3

This is number 4

This is number 5

 

... 称为闭区间运算符，用于定义从 x 到 y 的值范围，并包含范围中的 x 值和 y 值。有一个伴随运算符 (..<)，称为半开区间运算符。定义的范围从 x 到 y，但不包含 y 值。

在上面的代码中，index 常量用于自定义花括号内要执行的操作（for-in 循环）。第一次执行花括号内的语句时，index 的值为 1，即该范围内的第一个值。执行完成后，index 的值更新为 2，即该范围内的下一个值。随着 index 更新为每一个值，print 语句执行了五次。在循环遍历整个范围之后，循环完成，系统将继续执行循环之后的语句代码。

但是，假设结果不需要使用范围内的值。如果你只需要以特定次数执行一系列步骤，则可以跳过向常量赋值这一过程，并使用通配符 _ 来替换常量名称：

 

for _ in 1...3 {

print(”Hello!”)

}

控制台输出：

Hello!

Hello!

Hello!

 

你可以使用同样的 for-in 语法来迭代数组中的每一项。

 

let names = [”Joseph”, “Cathy”, “Winston”]

for name in names {

print(”Hello \(name)”)

}

控制台输出：

Hello Joseph

Hello Cathy

Hello Winston

 

由于 String 是集合类型，其功能类似于数组。你可以使用 for-in 循环来迭代 String 中的每个字符。

 

for letter in “ABCD” {

print(”The letter is \(letter)”)

}

控制台输出：

The letter is A

The letter is B

The letter is C

The letter is D

 

如果除了每个元素的值之外，还需要元素的索引，该怎么做？可以使用数组或字符串的 enumerated() 方法返回元组（元组是一种用括号括住的特殊类型，可用于显示的有序值列表），以显示每一项的索引和值：

 

for (index, letter) in “ABCD”.enumerated() {

print(”\(index): \(letter)”)

}

控制台输出：

0: A

1: B

2: C

3: D

 

也可以使用半开区间运算符实现相同的效果：

let animals = ["Lion", "Tiger", "Bear"]

for index in 0..<animals.count {

print("\(index): \(animals[index])")

}

控制台输出：

0: Lion

1: Tiger

2: Bear

 

如果将 for-in 循环用于字典，该循环生成的元组会显示每个条目的键和值。由于字典通常是通过指定某个键来访问，因此当采用循环来遍历字典时，无法保证各项将按特定的顺序来执行：

 

let vehicles = [”unicycle”: 1, “bicycle”: 2, “tricycle”: 3,

“quad bike”: 4]

for (vehicleName, wheelCount) in vehicles {

print(”A \(vehicleName) has \(wheelCount) wheels”)

}

控制台输出：

A unicycle has 1 wheels

A bicycle has 2 wheels

A tricycle has 3 wheels

A quad bike has 4 wheels

 

While 循环

在 while 循环中，如果指定条件不再为 true，即会停止执行。假设你希望一直畅玩游戏，直到耗尽所有“生命”。继续执行循环的条件即为生命数量大于 0：

 

var numberOfLives = 3

 

while numberOfLives > 0 {

playMove()

updateLivesCount()

}

 

Swift 会在每次执行循环之前检查条件，这意味着，如果从未满足条件，则可能会完全跳过该循环。如果在上面的示例中将 numberOfLives 初始化为 0，则 while 循环将确定 0 > 0 为 false，并且永远不会处理该循环的主体。

因此，while 循环的主体应该执行的是最终会改变条件的内容。在下面的示例中，numberOfLives 没有进行任何更新，因此条件始终解析为 true，并将永远执行下去。

 

var numberOfLives = 3

 

while numberOfLives > 0 {

print(”I still have \(numberOfLives) lives.”)

}

 

所以，主体所执行的语句应在某个时刻导致条件变为 false：

 

var numberOfLives = 3

var stillAlive = true

while stillAlive {

numberOfLives -= 1

if numberOfLives == 0 {

stillAlive = false

}

}

 

repeat-While 循环

repeat-while 循环类似于 while 循环，但它的语法会在检查条件之前执行一次代码块。

var steps = 0

let wall = 2 // 两步之后会有一堵墙

 

repeat {

print("Step")

 

steps += 1

 

if steps == wall {

print("You've hit a wall!")

break

}

} while steps < 10 // 这个方向的最大值

控制台输出：

Step

Step

You've hit a wall!

 

控制转移语句

在某些情况下，可能会希望停止执行循环主体内的循环。Swift 关键字 break 将中断循环内的代码执行，并开始执行在循环之后定义的代码（可通过 switch 语句识别出这个位置）。

在下面的代码中，如果 counter 达到 0，则循环中断：

 

// 打印 -3 到 0

for counter in -3...3 {

print(counter)

if counter == 0 {

break

}

}

控制台输出：

-3

-2

-1

0

 

在有些情况下，你可能会希望跳至循环中的下一个迭代。break 关键字会完全结束循环，而 continue 则会移至下一个迭代。例如，一个数组中的每个元素的类型都是 Person，你想要循环遍历该数组，并向年龄在 18 岁及以上的每个人发送电子邮件：

 

for person in people {

if person.age < 18 {

continue

}

 

sendEmail(to: person)

}

 

计算机十分擅长根据要求反复执行任务。无论是希望对集合进行迭代，还是重复执行一系列步骤直到满足某个条件，都需要熟悉循环这一重要概念。你将在编程工作中频繁使用循环。
