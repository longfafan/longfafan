在“Swift 和 Playground 简介”这节课中，你已经学习了 Swift 程序是在 .swift 文件中编写的，并按照从上到下的顺序执行。随着你使用 playground 文件进行工作，你将亲自见证这一点。不过，大型 app 并非如此简单。它们不适合都写在一个文件中，而且通常也不会从上到下逐行运行代码。

你将编写代码， 并根据先前执行的代码的结果来决定应该执行哪行代码。这称为“控制流程”。

在本节课中，你将学习如何使用逻辑运算符来检查条件，并使用控制流程语句（if、if-else 和 switch）来选择因此将执行哪些代码。

  • 如何使用逻辑运算符 NOT (！)、AND (&&) 和 OR (||) 来检查某个条件为 true 还是 false
  • 如何使用 switch 语句来控制执行哪些代码
  • 如何使用三元条件运算符 (?:) 有条件地将不同的值赋给常量或变量

想一个你使用过需要登录功能的 app 吧。如果应用程序已启动并已登录，那么 app 中会显示你的数据。如果尚未登录，那么 app 会要求你提供登录凭证。如果输入了正确的用户名和密码，就可以成功登录，并且 app 中会显示你的数据。如果输入的凭证不正确，那么会要求你输入正确的信息。

此示例描述了需要根据结果进行多次检查并运行代码的常见交互。

这些检查称为“条件语句”，它们是称为“控制流程”的更广义概念的一部分。作为开发者，你可以使用控制流程工具来检查某些条件，然后根据这些条件执行不同的代码块。

你可以根据一组条件的结果，使用各种语句来控制执行哪些代码。这些条件语句都是控制流程的示例。

你可以混合搭配运算符来创建布尔值或者 Bool 类型的值。布尔值为 true 或 false。if 语句中可以结合使用布尔值来确定应该运行代码还是跳过代码。

if 语句

if 语句是最直接的条件语句。if 语句的意思是“如果此条件为 true，则会运行此代码块”。如果该条件不为 true，程序会跳过该代码块。

在大多数情况下，你将使用 if 语句来检查只有几个可能结果的简单条件。以下是一个示例：

 

let temperature = 100

if temperature >= 100 {

print(”The water is boiling.”)

}

控制台输出：

The water is boiling.

 

temperature 常量等于 100。如果 temperature 大于或等于 100，则 if 语句打印文本。由于 if 语句解析为 true，所以会执行 if 语句伴随的代码块。

if-else 语句

你刚刚已经学习了如果条件为 true，if 语句会运行某个代码块。但是，如果条件不为 true，情况会怎样？通过给 if 语句添加 else 子句，你可以指定在条件不为 true 时执行某个代码块：

 

let temperature = 100

if temperature >= 100 {

print(”The water is boiling.”)

} else {

print(”The water is not boiling.”)

}

 

你可以进一步拓展这种想法。使用 else if 可以声明根据任意数量的条件来运行更多代码块。以下代码检查运动员在比赛中的位置并做出相应的反应：

 

var finishPosition = 2

 

if finishPosition == 1 {

print(”Congratulations, you won the gold medal!”)

} else if finishPosition == 2 {

print(”You came in second place, you won a silver medal!”)

} else {

print(”You did not win a gold or silver medal.”)

}

 

你可以使用多个 else if 语句来处理任意数量的可能情况。

布尔值

你可以将逻辑比较的结果赋给 Bool 常量或变量，以便稍后检查或访问该值。Bool 值只能是 true 或 false。

在以下代码中，Bool 语句确定 number 不符合 isSmallNumber：

 

let number = 1000

let isSmallNumber = number < 10

// number 不小于 10，因此 isSmallNumber 的值为 false

 

在这里，Bool 语句确定 currentSpeed 确实符合 isSpeeding。

 

let speedLimit = 65

let currentSpeed = 72

let isSpeeding = currentSpeed > speedLimit

// currentSpeed 大于 speedLimit，因此 isSpeeding 的值为 true

 

 

还可以使用以 ! 表示的 NOT 逻辑运算符来反转 Bool 值。

 

var isSnowing = false

 

if ! isSnowing {

print(”It is not snowing.”)

}

控制台输出：

It is not snowing.

 

同样，你可以使用以 && 表示的 AND 逻辑运算符来检查两个或多个条件是否为 true。

 

let temperature = 70

 

if temperature >= 65 && temperature <= 75 {

print(”The temperature is just right.”)

} else if temperature < 65 {

print(”It is too cold.”)

} else {

print(”It is too hot.”)

}

控制台输出：

The temperature is just right.

 

在上面的代码中，温度满足两个条件：大于或等于 65 度，并且小于或等于 75 度。这个值正合适。

现在你可以选择使用以 || 表示的 OR 逻辑运算符来检查两个条件中的任意一个条件是否为 true。

 

var isPluggedIn = false

var hasBatteryPower = true

 

if isPluggedIn || hasBatteryPower {

print(”You can use your laptop.”)

} else {

print(”😱”)

}

控制台输出：

You can use your laptop.

 

这个示例会打印“You can use your laptop.”。因为尽管 isPluggedIn 为 false，但是 hasBatteryPower 为 true，而 || OR 运算符确定其中一个选项为 true。

Switch 语句

你已了解可用于根据某些条件运行某些代码块的 if 语句和 if-else 语句。但是，如果使用一长串 if、else if、else if ... else 语句，那么代码可能在几个选项之后就会变得非常混乱而难以阅读。Swift 中还有一种称为 switch 语句的流程控制工具，这种语句非常适合处理多个可能条件。

基本 switch 语句允许一个值具有多个选项，可根据每个选项或 case 语句来分别运行代码。你还可以提供 default case 语句，指定将会针对所有尚未专门定义的情况而运行的代码块。

例如，要根据交通工具的车轮数量来打印交通工具的名称，可使用以下代码：

 

let numberOfWheels = 2

switch numberOfWheels {

case 0:

print("Missing something?")

case 1:

print(”Unicycle”)

case 2:

print(”Bicycle”)

case 3:

print(”Tricycle”)

case 4:

print(”Quadcycle”)

default:

print(”That’s a lot of wheels!”)

}

 

代码中给定常量 numberOfWheels，如果值为 0、1、2、3 或 4，则分别提供对应的响应动作。如果 numberOfWheels 为其他值，代码还提供了另一个响应动作。

你可以使用嵌套 if-else 语句来编写该代码，但是该代码很快会变得难以解析。

任意给定 case 语句都可以一次评估多个条件。例如，以下代码可检查某个字符是否为元音：

 

let character = “z”

 

switch character {

case “a”, “e”, “i”, “o”, “u”:

print(”This character is a vowel.”)

default:

print(”This character is a consonant.”)

}

使用数字时，可以使用区间匹配来检查数字是否在范围内。请参考以下代码片段中的语法：

 

switch distance {

case 0...9:

print(”Your destination is close.”)

case 10...99:

print(”Your destination is a medium distance from here.”)

case 100...999:

print(”Your destination is far from here.”)

default:

print(”Are you sure you want to travel this far?”)

}

 

如果希望根据多个不同条件运行不同代码，那么 switch 运算符就是最合适的流程控制工具。

三元条件运算符

if 语句的一个有趣且非常常见的用法是设置变量或返回值。如果某个条件为 true，则会希望将变量设置为某个值。如果该条件为 false，则会希望将该变量设置为其他值。

例如以下代码。该代码会检查某个数字的值是否大于另一个值，并会将较大的值赋给 largest 变量：

var largest: Int

 

let a = 15

let b = 4

 

if a > b {

largest = a

} else {

largest = b

}

由于这种情况在编程中十分常见，为了让编写的代码更简洁，许多语言都会而使用“三元条件运算符 (?:)”这种特殊运算符。Swift 程序员通常会省略“条件”一词，而直接称之为“三元运算符”。

“三元运算符”由三个部分组成：

  1. 答案为 true 或 false 的问题。
  2. 问题的答案为 true 时的值。
  3. 问题的答案为 false 时的值。

以下是一个示例：

 

var largest: Int

let a = 15

let b = 4

 

largest = a > b ? a : b

 

仔细看一下代码的最后一行。你可以这样理解：“如果 a > b，则会将 a 赋给 largest 变量；否则赋值 b”。在本例中，a 大于 b，所以将 a 赋给 largest。

在 Swift 中使用三元运算符并非必需。但是，根据条件进行赋值的方法，要比使用复杂的 if-else 语句实用得多。

提示

Swift 提供了一个全局函数，可使用一种更简明易懂的表达式查找最大值。

largest = max(a, b)
