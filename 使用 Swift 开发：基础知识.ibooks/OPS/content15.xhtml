在前面的课程中，你学习了利用现有函数，如 print 来执行任务。在那些示例中，函数都是现成的，你不用担心函数实现的细节是如何的。但要编写可维护、可重复使用的代码，则需要创建你自己的函数。

在本节课中，你将学习如何使用不同的参数和返回类型来声明函数，从而更好地理解抽象化的重要性。

内容提要

  • 如何编写自己的具有或没有返回类型的函数
  • 如何为你的函数指定输入参数
  • 如何在需要时返回多个值
  • 如何自定调用函数的方式
  • 如何为函数提供默认参数值

词汇

  • Argument label (自变量标签)
  • Parameter (参数)
  • Return type (返回类型)
  • Return value (返回值)

当有人给你下达了“穿上衣服”这个任务时，你可能会觉得这个指示非常简单。但实际上“穿上衣服”这个短语中包含了很多细节，比如要穿什么衣服、要如何穿上衣服，以及要从哪里获取衣服等。定义一个较为简单的方法对复杂的内容进行引用的想法即是抽象化，函数就是在代码中创建抽象化的重要方式之一。

函数由名称、一组输入和一组输出构成。输入和输出都是可选的。调用函数可以使程序执行某个操作，如将文本打印到控制台。

假设你要给朋友发送一条短信。“发送短信”就是要执行的操作。朋友的联系信息和要发送给朋友的短信内容就是参数。朋友是否收到短信就是返回值。

在 Swift 中，函数可以有 0 个、1 个或多个参数，类似地，也可以返回 0 个、1 个或多个值。函数有返回值时，你可以选择忽略值。

与用 let 声明常量或用 var 声明变量类似，func 关键字会告诉 Swift 编译器你正在声明一个函数。紧接着在 func 之后添加函数名称，然后添加圆括号 ()，圆括号内可以包含或不包含参数列表。如果函数有返回值，还需要添加一个箭头 (->)，如 Int、Sring、Person 等。

 

定义函数

func functionName (parameters) -> ReturnType {

// 函数主体

}

 

下面的函数示例将显示圆周率 (π) 的前十位数字。由于打印圆周率无需额外输入内容，因此函数没有参数。而且打印圆周率不需要在函数调用时返回任何相关信息，因此没有指定返回值。

 

func displayPi() {

print(”3.1415926535”)

}

 

在正确声明函数之后，就可以通过编写函数名称从任意位置调用或执行该函数。

 

displayPi()

控制台输出：

3.1415926535

 

参数

要指定带有参数的函数，请将值的名称、冒号 (:) 和值的类型全部插入到圆括号内。例如，假设你想编写一个称为 triple 的函数，此函数接受 Int，并会将 Int 值乘以三，再打印结果。

 

func triple(value: Int) {

let result = value * 3

print(”If you multiply \(value) by 3, you’ll get \(result).”)

}

 

在上例中，参数 value 是可以在函数中使用的常量。在调用函数时，可将其参数的值作为自变量传递。下面的代码是对函数 triple(value:) 的调用，其中 value 的自变量为 10。

 

triple(value: 10)

控制台输出：

If you multiply 10 by 3, you’ll get 30.

 

要为函数指定多个参数，请使用逗号 (,) 分隔各个参数。下面的函数示例将接受两个 Int 参数，使这二者相乘，然后打印结果：

 

func multiply(firstNumber: Int, secondNumber: Int) {

let result = firstNumber * secondNumber

print(”The result is \(result).”)

}

 

接下来，在函数调用中，为每个参数指定一个自变量：

 

multiply(firstNumber: 10, secondNumber: 5)

控制台输出：

The result is 50.

 

自变量标签

到目前为止，传递给函数的自变量的标签与函数在内部使用的参数名称相同。在下面的示例中，函数内部和函数调用中都使用了 firstname：

 

func sayHello(firstName: String) {

print(”Hello, \(firstName)!”)

}

 

sayHello(firstName: “Aidyn”)

 

不过假设想让自己的函数看起来更清晰，可以进行如下操作：

 

sayHello(to: “Miles”, and: “Riley”)

 

自变量标签 to 和 and 使函数看起来非常清晰：“say hello to Luke and Dave”。不过，请查看此函数的实现方式：

 

func sayHello(to: String, and: String) {

print(”Hello \(to) and \(and)”)

}

 

在函数的主体内，不建议使用 to 和 and 作为参数名称。要使函数中的参数名称不同于调用函数时使用的标签，请分别在参数名称之前指定自变量标签。在下面的代码中，to 是参数 person 的自变量标签，而 and 是参数 anotherPerson 的自变量标签。

 

func sayHello(to person: String, and anotherPerson: String) {

print(”Hello \(person) and \(anotherPerson)”)

}

 

sayHello(to: “Miles”, and: “Riley”)

 

如果没有自变量标签会让函数变得更加清晰，你可继续并忽略该标签。例如，以 print 函数为例：

 

print(”Hello, world!”)

 

函数的名称已经清楚说明了可以作为参数传递的内容。如果你一定要按以下方式来调用函数则显得有些画蛇添足：

 

print(message: “Hello, world!”)

 

要省略自变量标签，请使用 _。在下面的示例中，为了提高函数调用的可读性，省略了 firstNumber 参数的标签，但为 to 标签添加了含义：

 

func add(_ firstNumber: Int, to secondNumber: Int) -> Int {

return firstNumber + secondNumber

}

 

let total = add(14, to: 6)

 

默认参数值

参数可以增加函数的灵活性，但通常会有一个常用值。例如，你可能每餐都要喝水，但时不时也会用一杯橙汁来代替。

作为函数定义的一部分，你可以为任何参数提供一个默认值。这样你就可以调用带或不带参数的函数了。如果未指定参数，函数将直接使用默认值。

 

func display(teamName: String, score: Int = 0) {

print(”\(teamName): \(score)”)

}

 

display(teamName: “Wombats”, score: 100) // ”Wombats: 100”

display(teamName: “Wombats”) // ”Wombats: 0”

 

在使用默认参数值和自变量标签定义函数时，有一些需要注意的地方。编译器只能在一定的情况下推断出你要调用的内容。如果你将默认值和不带自变量标签的参数混在一起，就需要确保函数调用时每个参数传入的值都能够被准确地推断。

例如，按如下所示定义时，编译器无法推断提供的值要被哪个参数使用。

func displayTeam(_ teamName: String, _ teamCaptain: String = "TBA", _ hometown: String, score: Int = 0) {

// ...··

}

 

displayTeam("Dodgers", "LA") // 错误：调用参数 #3 时缺少参数

 

你可能觉得很明显，你是希望函数使用 teamCaptain 的默认值“TBA”，将“Dodgers”赋给 teamName，并将“LA”赋给 hometown。但编译器只能识别出三个不带自变量标签的 String 参数，以及两个要赋的值。它无法认定你打算使用 teamCaptain 的默认值。

你最好将使用默认值的自变量置于函数签名的末尾，并始终提供自变量标签。在上述示例中，如果 teamCaptain 具有自变量标签，编译器就能进行正确的推断。

func displayTeam(_ teamName: String, _ hometown: String, teamCaptain: String = "TBA", score: Int = 0)

 

返回值

有些时候，你可能不想在 multiply 函数所返回的结果之前打印出“The result is”。更合理的做法是函数只返回新值。为此，你将需要调整函数声明使其具有返回值，并指定值的类型。先前已介绍过，两个 Int 类型相乘结果始终是 Int，因此要使用的返回类型就是 Int。

 

func multiply(firstNumber: Int, secondNumber: Int) -> Int

 

在函数的主体内，使用 return 关键字来指定返回函数要返回的内容。在本例中，要返回的是 result：

 

func multiply(firstNumber: Int, secondNumber: Int) -> Int {

let result = firstNumber * secondNumber

return result

}

 

如果不想将两个值相乘，然后再将新值储存在 result 中，并立即返回该值，也可以在编写函数时不使用常量：

 

func multiply(firstNumber: Int, secondNumber: Int) -> Int {

return firstNumber * secondNumber

}

 

从 Swift 5.1 开始，如果函数只包含单行代码，那么还可以省略 return 关键字。multiply(firstNumber:secondNumber:) 具有返回类型，并且其主体中只有一行代码，因此该表达式的结果可用作其返回值。

func multiply(firstNumber: Int, secondNumber: Int) -> Int {

firstNumber * secondNumber

}

 

要调用此函数并使用返回值，请将返回值赋给常量：

 

let myResult = multiply(firstNumber: 10, secondNumber: 5)

//myResult = 50

print(”10 * 5 is \(myResult)”)

 

如果将来不需要再次使用 myResult，你可在 print 语句内使用函数，并跳过为常量赋值的步骤：

 

print(”10 * 5 is \(multiply(firstNumber: 10, secondNumber: 5))”)
