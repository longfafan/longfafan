作为程序员，你会面临很多需要从一定数量的选项中赋值的情况。假设你要编写一个程序，让乘客可以从三个选项中选择座位：靠窗、中间、靠过道。在 Swift 中，可以使用枚举来执行此操作。

枚举（或 enum）是可用于表示一组指定选项的特殊 Swift 类型。在本节课中，你将学习通常在何时使用枚举、如何定义枚举以及如何利用 switch 语句来处理枚举。

枚举定义一组相关值的共同类型。

例如指南针 app 上的方向：东、南、西、北。该 app 可以帮助用户确定这四个方向中的任意方向。指南针的指针始终指向北，但指南针的朝向随着用户的移动而移动。这可帮助用户确定他们面朝哪个方向。

使用关键字 enum 定义一个新枚举。下面的代码定义了一个用于跟踪指南针方向的 enum：

 

enum CompassPoint {

case north

case east

case south

case west

}

 

enum 定义了类型，case 选项定义了该类型允许的变量值。最好将枚举名称大写，将 case 选项小写。

此外还可以在一行中定义多个变量 case，以逗号分隔：

 

enum CompassPoint {

case north, east, south, west

}

 

定义了枚举之后，就可以开始像使用 Swift 中的其他任何类型一样使用枚举了。同时指定枚举类型和枚举值即可：

 

var compassHeading = CompassPoint.west

// 编译器将 compassHeading 的类型推断为 CompassPoint。

 

var compassHeading: CompassPoint = .west

// 编译器根据类型标注，将 compassHeading 的类型指定为 CompassPoint，然后使用点语法为其赋值。

 

既然已经设置 compassHeading 类型，那么接下来可以使用较短的点语法将值更改为指南针的其他指向了。

 

compassHeading = .north

 

控制流程

在控制流程课程中，你学习了如何使用 if 语句和 switch 语句来响应 Bool 值。在处理不同的枚举 case 时，可以使用相同的控制流程逻辑。

思考下面的代码，它可以打印不同的句子，具体取决于将哪个 CompassPoint 设置为 compassHeading 常量：

 

let compassHeading: CompassPoint = .west

 

switch compassHeading {

case .north:

print(”I am heading north”)

case .east:

print(”I am heading east”)

case .south:

print(”I am heading south”)

case .west:

print(”I am heading west”)

}

let compassHeading: CompassPoint = .west

 

if compassHeading == .west {

print(”I am heading west”)

}

类型安全的好处

在 Swift 中，枚举非常重要，因为可以通过枚举以类型安全的方式表示信息，如字符串或数字。

假设用一组数据来表示特定流派的电影。在学习枚举之前，你可能已经定义了类似以下代码的简单电影结构体：

 

struct Movie {

var name: String

var releaseYear: Int?

var genre: String

}

 

因为采用该定义，所以在设置流派时你会使用 String：

 

let movie = Movie(name: “Finding Dory”, releaseYear: 2016,

genre: “Aminated”)

 

你注意到此构造器中的问题了吗？

很多 Swift 开发者都会说，genre 是“泛字符串类型”，而不是“强类型”。他们的意思是，genre 容易出现 String 值可能会出现的各种错误，其中一种就是拼写错误。假设你编写了代码来获取流派为“Animated”的所有电影，那样就会漏掉 Finding Dory。

更好的做法是，从名为 Genre 的枚举中选择一个值执行给 genre。

enum Genre {

case animated, action, romance, documentary, biography,

thriller

}

 

struct Movie {

var name: String

var releaseYear: Int?

var genre: Genre

}

 

let movie = Movie(name: “Finding Dory”, releaseYear: 2016,

genre: .animated)

 

此代码出错的可能性要低得多。编译器会强制执行安全措施，要求你在初始化新电影时从 Genre 枚举中选择一个 case。

枚举是 Swift 中功能极其强大的工具。只要想以类型安全的方式来表示字符串或数字，就可以使用枚举。随着要使用的数据越来越复杂，你将继续学习关于枚举的更多高级功能。
