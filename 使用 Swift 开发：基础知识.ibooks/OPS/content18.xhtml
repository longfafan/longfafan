到目前为止在本课程中，你已经学习了处理单个对象，无论是常量或是变量。但是，有时候你会发现将一组对象作为一个整体来处理，可能更为合理。集合可用于一次引用多个对象。例如，可以通过使用一个常量来处理十二颗鸡蛋，而不用管理十二个单独的常量。在本节课中，你将学习 Swift 中提供的各种集合类型以及如何为程序选择合适的集合。

内容提要

  • 如何声明常量和变量集合
  • 如何从数组和字典中添加和移除值
  • 如何选择合适的集合类型

词汇

  • Array (数组)
  • Dictionary (字典)

相关资源

  • Swift 编程语言指南：Collection Types

 

Swift 定义了两种你会频繁使用的集合类型：数组和字典。每种类型都提供了一种与多个对象进行交互的独特方法。在学习过程中随着对不同类型的了解的加深，你会注意到这些类型共享某些功能：在集合中添加/移除项、访问单个项，以及提供关于集合中数据的类型信息。

 

数组

数组是 Swift 中最常见的集合类型，是储存同一类型值的有序列表。当你声明数组时，可以指定集合将持有何种类型的值，或者也可以使用类型推断系统来确定类型。

通常会使用类似 Int 或 String 的字面量对数组进行初始化。要声明数组字面量，请用方括号将值的集合括起来，并用逗号分隔这些值。

 

[value1, value2, value3]

 

你将使用类似的语法来声明储存字符串的数组。注意在以下示例中使用方括号括起来的变量类型 String：

 

var names: [String] = [”Anne”, “Gary”, “Keith”]

 

由于类型推断能够确定“Anne”、“Gary”和“Keith”为字符串，所以实际可以跳过这一步直接声明数组而无需指定数组的类型：

 

var names = [”Anne”, “Gary”, “Keith”]

 

不过尽管类型推断可以确定类型，在某些情况下还是希望指定数组的类型。假设你希望集合由 8 位整数 (介于 -127 到 128 之间的数字) 组成。首先，将一个变量赋给以下数字集合：

 

var numbers = [1, -3, 50, 72, -95, 115]

 

Swift 将推断所有值均为 Int 类型，并会将 numbers 设置为整数数组或 [Int]。虽然这一推断确实是正确的，但是还有一个问题：Int 可以容纳不在 -127 到 128 这一范围内的正数和负数。要帮助 Swift 理解你希望将数组限制为较小的整数，可以指定类型为 [Int8]。这是一个由 8 位整数组成的数组，而且值仅限于前面提及的范围：

 

var numbers: [Int8] = [1, -3, 50, 72, -95, 115]

 

如果尝试在 [Int8] 类型的数组字面量中包含如 300 之类的较大数字，情况会怎样？Swift 会返回错误，因为 300 大于 Int8 的最大数值。如果推断的类型不够具体，则应指定类型，这样有助于将不允许的值排除在外。

检查数组中是否存在某个值，通常是非常有用的操作。要达成此目的，可以使用 contains(_:) 方法，向其传入期望的值。如果数组中包含该值，则表达式为 true，否则为 false：

 

let numbers = [4, 5, 6]

if numbers.contains(5) {

print(”There is a 5”)

}

 

在本课程先前的部分，你了解到常量在声明之后不可修改。当你使用 let 将集合赋给常量时，情况也是如此：你不能在集合内添加、移除或修改任何项。然而，如果使用 var 将集合储存在变量中，则将可以在该集合内添加、移除或修改项。此外，还将可以清空集合或将变量整个设置为不同的集合。

数组类型

数组就像一个篮子：它开始的时候可以为空，随后你可以用值来填充篮子。但是如果数组字面量不包含任何值，那么如何推断其类型？

你可以使用类型标注、集合类型标注或数组构造器来声明数组的类型。

此示例使用传统类型标注来定义数组。

 

var myArray: [Int] = []

 

此示例使用特殊集合类型标注来定义数组。你应该熟悉这种不太常见的做法，以防在你处理的代码中遇到这种情况。

 

var myArray: Array<Int> = []

 

就像在类型名称之后添加 () 就能对所有对象进行初始化一样，可以在 [Int] 之后添加 () 来对 Int 对象的空数组进行初始化。你也应该熟悉这一代码，以防在将来遇到它。

 

var myArray = [Int]()

 

使用数组

你会发现在某些情况下使用数组字面量枯燥乏味或容易出错。例如，假设你希望数组由 100 个 0 来填充。如果使用数组字面量的形式，则 100 个 0 全部都需要输入，这样做很容易数错数量。

没关系，Swift 有解决方案。不必数完 100 个 0，你可以使用以下构造器创建数组，使用重复默认值来计数 100 次：

 

var myArray = [Int](repeating: 0, count: 100)

 

要弄清楚数组内包含的项数，可以使用其 count 属性。如需检查数组是否为空，该怎么做？此时无需比较 count 与 0，你可以检查数组的 isEmpty 属性，该属性会返回 Bool 值：

 

let count = myArray.count

if myArray.isEmpty { }

 

在定义数组之后，可以使用多种方法和属性对其进行访问或修改。还可以使用数组的下标语法。下标语法在集合之后使用方括号来引用集合中的特定元素或范围。要从集合中检索特定值，请在方括号中指定希望访问的值的索引。数组的值始终从零开始索引，这意味着数组中第一个元素的索引为 0。以下示例将会检索名字集合中的第一个值：

 

let firstName = names[0]

 

你还可以使用下标语法来更改特定索引处的值：通过将下标放在 = 符号的左侧，可以设置特定值，而不是访问当前值。在下面的示例中，会将集合中的第二个名字更改为“Paul”。

 

names[1] = “Paul”

 

对数组使用下标索引时，需要确保你指定的索引在数组中存在。例如，如果指定 3 作为索引，数组必须至少有四个元素。如果没有，程序在执行该代码行时将崩溃。

在定义数组并设置某些值之后，通常会希望追加新值。你可以使用变量数组的 append 函数向数组末尾添加新元素。要一次追加多个元素，请使用 += 运算符。

 

var names = [”Amy”]

names.append(”Joe”)

names += [”Keith”, “Jane”]

print(names) // [”Amy”, “Joe”, “Keith”, “Jane”]

 

如果不希望在数组末尾追加新元素，该怎么做？变量数组还有一个 insert(_: at:) 函数，可在追加新元素时指定值和索引。在此场景中，正如使用下标时那样，你需要确保提供有效的索引。

由于数组从零开始索引，因此第一个元素的索引始终是 0，而最后一个元素的索引则等于 count–1。在以下示例中，会在数组的起始位置插入“Bob”：

 

names.insert(”Bob”, at: 0)

 

类似地，remove(at:) 函数用于移除指定索引处的项目。这一函数与 insert(_: at:) 不同，你不需要输入值；如下面的示例所示，该函数在默认情况下会返回移除的项目。另一种方法是 removeLast() 函数，此函数会从数组中移除最后一个项目而无需计算索引。最后，removeAll() 方法会从数组中移除所有元素。

 

var names = [”Amy”, “Brad”, “Chelsea”, “Dan”]

let chelsea = names.remove(at:2)

let dan = names.removeLast()

names.removeAll()

 

Swift 还允许通过将现有的两个同一类型的数组添加到一起来创建新数组。你可以指定在生成的数组中哪个数组先出现：

 

var myNewArray = firstArray + secondArray

 

那么数组中的数组呢？使用数组字面量语法可将两个数组放在另一个包容性数组中。然后可以针对包容性数组使用下标语法来访问嵌套数组中的其中一个数组。要访问其中一个嵌套数组中的特定元素，可以使用两组下标。

 

let array1 = [1,2,3]

let array2 = [4,5,6]

let containerArray = [array1, array2] //[ [1,2,3], [4,5,6] ]

let firstArray = containerArray[0] //[1,2,3]

let firstElement = containerArray[0][0] //1

 

字典

Swift 中的另一种集合类型就是字典。我们常说的字典由一系列单词及其释义组成。Swift 字典与此类似，由一系列键及其关联的值组成。就像字典中的每个单词都是唯一的一样，每个键也必须是唯一的。与英语字典按照字母顺序列出单词以便查找一样，Swift 字典经过优化处理可以快速进行键查找。

可以使用字典字面量来设置字典：用方括号将以逗号分隔的键/值对括起来。同时采用冒号来分隔每个键及其结果值：

 

[key1 : value1, key2: value2, key3: value3]

 

与数组一样，可以根据字典字面量中使用的类型来推断字典的类型。假设你希望储存游戏中的高分列表。你会使用玩家的名字作为键，使用玩家的分数作为相应的值。字典的类型将推断为键的类型为 String 且值的类型为 Int。这可以表示为 [String: Int] 或 Dictionary<String, Int>：

 

var scores = [”Richard”: 500, “Luke”: 400, “Cheryl”: 800]

 

与其他集合类型一样，字典有一个 count 属性可确定键-值对的数量，有一个 isEmpty 属性可确定字典是否没有任何键-值对。现在你可能熟悉创建空字典的语法了：

 

var myDictionary = [String: Int]()

 

var myDictionary = Dictionary<String, Int>()

 

var myDictionary: [String: Int] = [:]

 

所有这些示例都会产生相同类型的空字典。你可能会有经常使用的首选方法，但也应该熟悉所有方法。注意最后一个示例中空字典字面量的语法。

 

添加/移除/修改字典

下标语法特别适合处理字典。由于 Swift 字典中的顺序不重要，所以没有索引，也不存在与索引关联的下标错误风险。

以下示例向高分字典中添加了新分数。如果字典中已经存在“Oli”键，则此代码将会使用新值替换旧值。

 

scores[”Oli”] = 399

 

但是，如果希望在替换之前知道字典中是否存在旧值，该怎么做？如果存在旧值，则可以使用 updateValue(_:, forKey:) 来更新字典，该方法返回的值将与旧值相同。在以下示例中，更新之前，oldValue 与 Richard 的旧值相同。如果不存在值，则 oldValue 将为 nil。后面你将学习有关 nil 关键字的更多信息。它是表示不存在某个值的一种特殊方式。

 

let oldValue = scores.updateValue(100, forKey: “Richard”)

 

仅当该方法返回值时，Swift 才会使用 if-let 语法让你运行代码。如果不存在现有值，则将不会执行花括号内的代码：

 

if let oldValue = scores.updateValue(100, forKey: “Richard”) {

print(”Richard’s old value was \(oldValue)”)

}

 

要从字典中移除项目，则可以使用下标语法将值设置为 nil。与更新值类似，如果需要在移除旧值之前返回该值，则字典有 removeValue(forKey:) 方法可供使用：

 

var scores = [”Richard”: 100, “Luke”: 400, “Cheryl”: 800]

scores[”Richard”] = nil // [”Luke”: 400, “Cheryl”: 800]

 

if let removedValue = scores.removeValue(forKey: “Luke”) {

print(”Luke’s score was \(removedValue) before he stopped

playing”)

}

 

访问字典

Swift 字典提供了其他集合类型未包含的两种属性。你可以使用 keys 返回字典中所有键的列表，还可以使用 values 返回所有值的列表。如果你希望随后使用这些集合，则需要将其转换为数组：

 

var scores = [”Richard”: 500, “Luke”: 400, “Cheryl”: 800]

 

let players = Array(scores.keys) // [”Richard”, “Luke”, “Cheryl”]

let points = Array(scores.values) //[500, 400, 800]

 

要查找字典中的特定值，请使用 if-let 语法。如果字典中存在指定的键，则结果会是该键对应的值。然而，如果字典中不存在该键，则不会执行括号内的代码。

 

if let lukesScore = scores[”Luke”] {

print(lukesScore)

}

 

if let henrysScore = scores[”Henry”] {

print(henrysScore) // not executed; “Henry” is not a key in

the dictionary// 未执行：“Henry”不是字典中的键

}

控制台输出：

400
