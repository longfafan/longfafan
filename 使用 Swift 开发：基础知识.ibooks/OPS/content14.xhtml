在前面的课程中，你打印了标志性的“Hello, world!”字符串。你通过将一系列字符连串起来创建了“Hello, world!”。Swift 的字符串以 String 类型表示，最常用的定义方式是使用字符串字面量。

字符串字面量是 String 值的原始表示形式。字符串字面量的书写方式是使用双引号 " 括住一组字符。

字符串字面量常用于为常量或变量设置初始值。

 

let greeting = “Hello”

var otherGreeting = “Salutations”

 

如果将字符串赋给常量（使用 let），则该字符串是不可变的，无法修改。如果将字符串赋给变量（使用 var），则该字符串是可变的，可以更改。

如果字符串字面量需要多行，只需用三个双引号 “”” 括住字符。多行字符串字面量包含左右引号之间的所有行。字符串从左引号后的第一行开始，到右引号前的那一行结束。

 

let joke = """

Q: Why did the chicken cross the road?

A: To get to the other side!

"""

print(joke)

控制台输出：

Q: Why did the chicken cross the road?

A: To get to the other side!

 

在多行字符串字面量中，如果一行文本与结尾的 """ 引号对齐，则会忽略这一行前面的原有空格。多行字符串字面量可以包含双引号 (")。

let greeting = """

It is traditional in programming to print "Hello, world!"

"""

要在单行字符串字面量中包含双引号，需要使用反斜杠 (\)，这在 Swift 中称为转义符。这是因为你要对字符串中字符的正常解释进行“转义”。

 

let greeting = “It is traditional in programming to print

\”Hello, world!\””

 

可使用转义符搭配其他字母和符号来生成特定的结果：

  • 双引号：\”
  • 单引号：\’
  • 反斜杠：\\
  • 制表符：\t
  • 换行（转至下一行，类似于按 return）：\n

你会发现你通常希望从空字符串开始，然后再慢慢添加内容。这意味着你需要让字符串成为变量。使用下列语法，可以将字符串初始化为不包含任何文本：

 

var myString = “”

 

如需查看 Swift String 是否为空，可以使用布尔属性 isEmpty：

 

var myString = “”

 

if myString.isEmpty {

print(”The string is empty”)

}

 

你可能预料到了，单个字符属于 Character 类型。但较单个字符而言，字符串在编程中更为常见。因此，Swift 通常会将多个字符的组合甚至一个字符推断为 String，除非你使用类型标注另行指定：

 

let a = "a" // a 是字符串

let b: Character = “b” // b 是字符

 

串联和插值

字符串有时需要串联起来。+ 运算符不仅适用于数字，还可将字符串连接起来。可以使用 + 将多个 String 值连串起来创建一个新的 String 值。以上操作称为串联。

 

let string1 = “Hello”

let string2 = “, world!”

let myString = string1 + string2 // “Hello, world!”

 

如果现有 String 为变量，则可用 += 运算符进行添加或修改。

 

var myString = “Hello”

myString = myString + “, world!” // “Hello, world!”

myString += “ Hello!” // “Hello, world! Hello!”

 

随着字符串变得愈加复杂，使用 + 运算符可能会让你的代码变得棘手，难以处理。比如在上述代码中，你可能会忘记在“Hello!”前加一个空格。

Swift 提供了一种名为字符串插值的语法，可以让你更加轻松地在代码中包含常量、变量、字面量和表达式。字符串插值可用于轻松地将多个值合并到一个 String 常量或变量中。

要将常量或变量原始值插入到 String 中，可以在名称前面添加一个反斜杠 \，然后再以圆括号 () 括住该名称。在下面的示例中，打印的 String 会包含 name 和 age 常量的原始值。

 

let name = “Rick”

let age = 30

print(”\(name) is \(age) years old”) // Rick is 30 years old

 

你可以将整个表达式放在圆括号内。表达式会在计算之后再进行打印和储存。

 

let a = 4

let b = 5

print(”If a is \(a) and b is \(b), then a + b equals \(a+b)”)

控制台输出：

If a is 4 and b is 5, then a + b equals 9

 

使用字符串插值可将 a 和 b 常量的原始值插入到打印的 String 值中。

根据上文的提示，字符串插值的结果是 String，因此还可以将其用于任何要使用 String 或字符串字面量的地方：

let listName = "Shopping"

var items = 14

myLabel.text = "There are \(items) items on your \(listName) list"

// 这个标签文本会显示 “There are 14 items on your Shopping list”

 

func setLabel(_ label: UILabel, to text: String) {

label.text = text

}

 

setLabel(myLabel, to: "There are \(items) items on your

\(listName) list")

 

字符串相等性和比较

开发者需要经常比较 String 值来确定它们是否相等。

与处理数字的方式一样，可以使用 == 运算符来检查两个字符串是否相等。如你所料，== 会检查字符串内的字符及其顺序是否完全相同。由于大写字符和对应的小写字符并不相同，因此只有在每个字符的大小写都匹配的情况下，字符串才会具有相同值。这称为区分大小写。

 

let month = “January”

let otherMonth = “January”

let lowercaseMonth = “january”

if month == otherMonth {

print(”They are the same”)

}

 

if month ! = lowercaseMonth {

print(”They are not the same.”)

}

控制台输出：

They are the same.

They are not the same.

 

不过你可能希望在检查字符串相等性时忽略字符串的大小写。你可以使用 lowercased() 方法对两个字符串进行标准化，将字符串的全小写版本与调用字符串的全小写版本进行比较。

 

let name = “Johnny Appleseed”

if name.lowercased() == “joHnnY aPPleseeD”.lowercased() {

print(”The two names are equal.”)

}

控制台输出：

The two names are equal.

 

你也可以使用 lowercased() 的对应方法 uppercased()，这个方法会创建字符串的全大写版本。

如需匹配字符串的开头或结尾，可以使用 hasPrefix(_:) 或 hasSuffix(_:) 方法。就像 = = 一样，这两种方法也区分大小写。

 

let greeting = “Hello, world!”

print(greeting.hasPrefix(”Hello”))

print(greeting.hasSuffix(”world!”))

print(greeting.hasSuffix(”World!”))

控制台输出：

true

true

false

 

也许你想检查一个字符串是否包含在另一个字符串之内。可以使用 contains(_:) 方法返回一个布尔值，用于指示是否找到子字符串。

 

let greeting = “Hi Rick, my name is Amy.”

if greeting.contains(”my name is”) {

print(”Making an introduction”)

}

 

由于字符串是字符集合，因此字符串的长度等于字符总数。任何集合的大小都可以使用其 count 属性来确定。这一属性可用来比较字符串或判定字符串是否满足某种要求。

 

let name = “Ryan Mears”

let count = name.count // 10

 

let newPassword = “1234”

 

if newPassword.count < 8 {

print(”This password is too short.Passwords should have at

least eight characters.”)

}

控制台输出：

This password is too short.Passwords should have at least

eight characters.

 

上一单元中，你学习了如何根据特定情况使用 switch 语句来执行特定代码块。你还可以使用 switch 语句对多个字符串值或字符值进行模式匹配，并做出相应的响应。

 

let someCharacter: Character = “e”

switch someCharacter {

case “a”, “e”, “i”, “o”, “u”:

print(”\(someCharacter) is a vowel.”)

default:

print(”\(someCharacter) is not a vowel.”)

}

控制台输出：

e is a vowel.

Unicode

所有 Swift String 都遵循称为“Unicode”的国际计算标准。遵守 Unicode 让 Swift 突破了英语语言中字符和字母不够丰富的限制。而 Unicode 则涵盖了多种语言中使用的超过 128,000 个不同字符。这些字符包括字符变音符号 (é)、表情符号 (🐮)、符号 (∞)、日文汉字（七），以及其他特殊字符。此外，Unicode 还支持从右向左阅读的文本和从左向右阅读的文本。

Swift 确保你可以方便地使用 Unicode 字符，以便“e”、“é”和“🐮”被视为长度为 1 的单个字符。

 

let cow = "🐮"

let credentials = "résumé"

print("∞".count) // 1

 

实际上，一个字符是一个 Unicode“扩展字位群集”。这种说法听起来很复杂，其实就是表达，一些字符实际上是由多个（通常看不见的）字符构成的，只不过对于用户显示为单个字符。例如，“ü”由两个字符构成，而“👩🏽‍🎓”由四个字符构成！Swift 可以让这些字符更易于使用。
