在本节课中，你将学习如何创建一款基本的 iOS app，借此来熟悉 Xcode 的界面及其功能。

Xcode 文件类型

Xcode 知道如何跨多种编程语言来处理各种文件。现在，你将了解与使用 Swift 语言编写的项目相关的文件。

在项目导航器的最顶端，你将看到一个带有蓝色 Xcode 小图标的文件。点按该文件，并在编辑器区域打开。这是 .xcodeproj 文件，其中包含针对项目及其目标的所有设置。每个目标都是 Xcode 可以通过项目构建的产品。现在，你要构建的目标将是可执行的 app。稍后，可以使用该目标来构建框架、特定 app 的不同版本，或者用于诸如 watchOS 或 Apple TVOS 之类的不同平台的版本。

 

正如你所预期的那样，具有 .swift 扩展名的文件中包含 Swift 代码。无论何时构建 app，Xcode 都会收集所有提供的 .swift 文件，并通过 Swift 编译器运行这些文件，这一操作会将 Swift 代码转换为所选设备可以理解的格式。

对于 Interface Builder 来说，所有 .storyboard 文件都是唯一的。其中包含关于应用程序中每个场景的设计，以及关于某个屏幕过渡到另一个屏幕的方式的信息。在接下来的课程中，你将进一步了解关于 Storyboard 文件的更多内容。

.xcassets 文件代表素材目录。在素材目录中，可以管理许多不同种类的素材。这包括 app 的图标、图像、颜色定义以及要随 app 打包在一起的其他形式的数据。在素材目录中，还可以根据以下设备设置和功能，为素材指定变体：浅色和深色外观、高低对比度的辅助功能设置、从屏幕分辨率到内存容量和图形芯片支持的硬件差异等。

Info.plist 文件包含针对你的 app 的属性列表和设置。Xcode 提供了用于编辑此文件的特殊界面，这样你基本上就不需要直接与之进行交互。这些设置被安排在不同的屏幕上，你可以在选择 .xcodeproj 文件（先前已介绍过）时找到它们。

键盘快捷键

随着对 Xcode 越来越熟练，你会发现使用键盘快捷键来执行任务可大幅提升速度。请务必尽快学会最常用的快捷键：

  • command-B：构建项目
  • command-R：构建并运行项目
  • command-.：停止构建或运行
  • command-/：注释/取消注释选定代码行
  • command-[：左移选定代码
  • command-]：右移选定代码
  • control-I：缩进所选代码
  • command-0：显示和隐藏导航器
  • option-command-0：显示和隐藏检查器

要了解更多键盘快捷键，请查看任意菜单的右侧，或从菜单栏中选择“Xcode”>“Preferences”，然后在窗口顶部选择“Key Bindings”。

Xcode 偏好设置

Xcode 是一款功能强大的工具，其中包含许多选项。当你使用 Xcode 时，你将进一步了解关于 Xcode 的更多功能，以及如何自定该程序以达到你想要的效果。要查看 Xcode 偏好设置，可以从菜单栏中选择“Xcode”>“Preferences”。你可以使用“Preferences”添加开发者账户、自定导航或字体，或在进行文本编辑时选择某些行为等等。

打开 Xcode 的“Preferences”，然后导航到“Themes”窗格。

可以通过在左侧的列表中选择不同的选项来选择一个新主题，或者一组颜色和字体。有些主题具有“(Light)”或“(Dark)”后缀。这些变体会根据系统当前是浅色外观还是深色外观而相应地进行变化。要更改 Xcode 的这种外观行为，使其独立于系统设置，可以使用 Xcode 的“General”窗格来完成。

要阅读有关 Xcode 及其各种工具的更多信息，请从菜单栏中选择“Help”>“Xcode Help”，以阅读 Xcode Help 文档。
